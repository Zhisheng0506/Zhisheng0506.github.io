1 ≤ n ≤ 20，序列长度 < 10（数字 0~9，无重复）。

⚠️ 关键坑点

1. 三个判断顺序不能乱

必须先判空，再比值。nullptr->val 是未定义行为。

2. 全空 = 相同

两棵空树是完全相同的。这是递归的终止条件——叶子节点的左右孩子都是 null，这里判断通过。

3. 左右子树都要相同

&& 保证左右都相同才返回 true，任意一侧不同就短路返回 false。

总结

要点 说明
核心 两棵树同时递归，逐节点比较
都空 结构一致到底 → true
一空一非空 结构不同 → false
值不同 内容不同 → false
值相同 递归比较左右子树，必须都相同
顺序 先判空 → 再判结构 → 最后比值
