## 题目描述

> 题目链接：[N诺/DreamJudge 1250](https://noobdream.com/DreamJudge/Issue/page/1250/)

输入一个 N×N 矩阵，判断是否对称（即 $A[i][j] = A[j][i]$）。

题目本身很简单，但这道题的代码里藏了一个很实用的技巧：**嵌套循环中如何快速跳出**。

<!--more-->

## 对称矩阵的判断逻辑

只需要检查下三角，看 $A[i][j]$ 和 $A[j][i]$ 是否相等，忽略对角线（$i = j$ 时必然相等）。

```cpp
int flag = 0;
for (int i = 0; i < n && !flag; i++) {     // ⚠️ !flag 控制外层
    for (int j = 0; j < n && !flag; j++) {  // ⚠️ !flag 控制内层
        if (i != j && N[i][j] != N[j][i]) {
            flag = 1;   // 标记后两层循环同时退出
        }
    }
}
```

一旦 `flag = 1`，内外层循环条件里的 `!flag` 立即为 false，两层同时终止。这是竞赛中最常用的写法。

## 嵌套循环跳出的四种写法

### 方法 1：标志变量（flag）

```cpp
bool flag = true;
for (int i = 0; i < n && flag; i++) {
    for (int j = 0; j < n && flag; j++) {
        if (N[i][j] != N[j][i]) {
            flag = false;
        }
    }
}
```

| 优点 | 缺点 |
|------|------|
| 代码直观，一看就懂 | 需要额外变量 |
| 多个退出条件也能统一 | flag 命名不好时语义模糊 |

**适用：日常开发首选，机试快速写。**

### 方法 2：函数返回

```cpp
bool isSymmetric(vector<vector<int>>& N) {
    int n = N.size();
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            if (i != j && N[i][j] != N[j][i]) {
                return false;  // ⚠️ 直接 return，瞬间跳出
            }
        }
    }
    return true;
}
```

| 优点 | 缺点 |
|------|------|
| 最清晰，`return` 瞬间结束 | 需要单独写函数 |
| 无额外变量 | 逻辑太长时函数拆分会变多 |

**适用：逻辑可复用、函数不长时最优雅。**

### 方法 3：lambda + return（C++11）

```cpp
bool result = [&]() {
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            if (i != j && N[i][j] != N[j][i]) {
                return false;
            }
        }
    }
    return true;
}();
```

| 优点 | 缺点 |
|------|------|
| 不用单独命名函数 | lambda 语法不够直观 |
| return 直接跳出 | 多了一层 `()` 调用 |

**适用：逻辑只用一次不值得命名函数，又想用 return 跳出。**

### 方法 4：goto（慎用）

```cpp
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        if (N[i][j] != N[j][i]) {
            goto not_symmetric;
        }
    }
}
not_symmetric:
// 跳出后统一处理
```

| 优点 | 缺点 |
|------|------|
| 直接跳转，嵌套越深优势越大 | 破坏结构，被广泛诟病 |
| 从 ≥3 层嵌套跳出最方便 | 维护时容易出问题 |

**适用：内核代码或嵌套 ≥3 层且性能敏感时。机试不需要。**

---

## 怎么选

| 场景 | 推荐 |
|------|------|
| 机试快速写 | 标志变量 |
| 逻辑可复用 | 函数 return |
| 只用一次但想 return | lambda |
| 嵌套 ≥3 层 | goto，但建议先简化嵌套 |

对于机试，**标志变量**和**函数 return** 就够用了，99% 的题不会超过两层循环。

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n;
    while (cin >> n) {
        vector<vector<int>> N(n, vector<int>(n));
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                cin >> N[i][j];
            }
        }

        int flag = 0;
        for (int i = 0; i < n && !flag; i++) {
            for (int j = 0; j < n && !flag; j++) {
                if (i != j && N[i][j] != N[j][i]) {
                    flag = 1;
                }
            }
        }

        if (flag) {
            cout << "No!" << endl;
        } else {
            cout << "Yes!" << endl;
        }
    }
}
```
