二叉树-前中序推后序

Posted on 2026-06-13 Edited on 2026-06-26

题目描述

题目链接：N诺/DreamJudge 1300

给定二叉树的前序遍历和中序遍历，求后序遍历。

n ≤ 26，结点以大写字命名。

核心思路

前序确定根，中序划分左右子树。

算法拆解

图解

为什么前序+中序能唯一确定一棵树

  • 前序第一个确定根节点
  • 中序中根的位置确定左右子树的节点集合
  • 左右子树又是同样的结构 → 递归

前序+后序不能唯一确定（缺少左右子树的分界信息）。但前序+中序、中序+后序都可以。

⚠️ 关键坑点

1. 前序切分时的偏移

前序的布局是 [根][左子树][右子树]。左前序跳过根（下标 1），右前序跳过根+左子树。

2. find 返回值是 size_t

用 int 更安全，避免 size_t 和负数比较时的隐式转换问题。

3. 空串终止条件

叶子节点递归下去，左右子树为空，返回空串。

总结

要点 说明
核心 前序首字符 = 根，中序根位置分左右
前序布局 [根][左子树][右子树]
切分 左 pre 从 1 开始，右 pre 跳过根+左子树长度
递归公式 后序 = 左后序 + 右后序 + 根
终止条件 pre.empty() → ""
