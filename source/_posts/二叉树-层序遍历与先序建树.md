## 题目描述

> 题目链接：[N诺/DreamJudge 1364](https://noobdream.com/DreamJudge/Issue/page/1364/)

按先序遍历序列建立二叉链存储的二叉树（`0` 表示空节点），实现先序、中序、后序、层序遍历，并计算叶子结点个数。

```
输入: A B C 0 0 0 D E 0 F 0 0 G 0 0

对应树:
        A
       / \
      B   D
     /   / \
    C   E   G
         \
          F

输出:
先序: A B C D E F G
中序: C B A E F D G
后序: C B F E G D A
叶子: 3  (C, F, G)
```

三种递归遍历在 [二叉树-前中后序遍历模板](/2026/06/04/二叉树-前中后遍历/) 已讲过，这里一笔带过。本篇重点在**先序建树**、**层序遍历**和**数叶子**。

## 先序遍历序列建树（重点）

```
输入序列: A B C 0 0 0 D E 0 F 0 0 G 0 0

'A' → 建节点 A，递归建左子树
  'B' → 建节点 B，递归建左子树
    'C' → 建节点 C，递归建左子树
      '0' → 返回 null（C 的左孩子为空）
      '0' → 返回 null（C 的右孩子为空）
    C 完成，返回
  B 完成，递归建右子树
    '0' → 返回 null（B 的右孩子为空）
  B 完成
A 的左子树完成，递归建右子树
  'D' → 建节点 D，递归建左子树
    'E' → 建节点 E，递归建左子树
      '0' → null
      递归建右子树
        'F' → 建节点 F
          '0' → null
          '0' → null
      F 完成
    E 完成
  D 的左子树完成，递归建右子树
    'G' → 建节点 G
      '0' → null
      '0' → null
    G 完成
  D 完成
A 完成
```

### 代码

```cpp
Node* buildtree(vector<char>& A, int& index) {
    if (index >= A.size()) return nullptr;

    char x = A[index++];         // 取出当前字符，指针后移
    if (x == '0') return nullptr; // 0 = 空节点

    Node* root = new Node(x);
    root->left  = buildtree(A, index);  // 递归建左子树
    root->right = buildtree(A, index);  // 递归建右子树
    return root;
}
```

**关键**：`index` 是**引用**，在递归过程中全局前进。先序建树的顺序就是"根 → 左子树 → 右子树"，和先序遍历完全一致。

## 层序遍历（重点）

层序就是**一层一层地从上到下、从左到右**访问。用**队列**实现：

```cpp
void levelorder(Node* root) {
    if (root == nullptr) return;
    queue<Node*> q;
    q.push(root);
    while (!q.empty()) {
        Node* cur = q.front(); q.pop();
        cout << cur->val << " ";
        if (cur->left)  q.push(cur->left);
        if (cur->right) q.push(cur->right);
    }
}
```

### 执行过程

```
        A          队列: [A]
       / \
      B   D

输出 A → 左B右D入队 → 队列: [B, D]
输出 B → 左C右null入队 → 队列: [D, C]
输出 D → 左E右G入队 → 队列: [C, E, G]
输出 C → 无孩子 → 队列: [E, G]
输出 E → 左null右F入队 → 队列: [G, F]
输出 G → 无孩子 → 队列: [F]
输出 F → 无孩子 → 队列: []

输出: A B D C E G F
```

### 层序 vs 前序

```
前序: A B C D E F G   ← 深度优先（递归）
层序: A B D C E G F   ← 广度优先（队列）
```

## 数叶子节点

叶子 = 左右孩子都为 `nullptr` 的节点：

```cpp
int countleaves(Node* root) {
    if (root == nullptr) return 0;
    if (root->left == nullptr && root->right == nullptr) {
        return 1;   // 是叶子
    }
    return countleaves(root->left) + countleaves(root->right);
}
```

逻辑：空树 → 0；自己是叶子 → 1；否则 = 左子树叶数 + 右子树叶数。

```
叶子:
  C: 左右都为 null → 叶子 ✓
  F: 左右都为 null → 叶子 ✓
  G: 左右都为 null → 叶子 ✓
总: 1+1+1 = 3
```

## 递归遍历（简）

```cpp
void preorder(Node* root) {
    if (root == nullptr) return;
    cout << root->val << " ";
    preorder(root->left);
    preorder(root->right);
}

void inorder(Node* root) {
    if (root == nullptr) return;
    inorder(root->left);
    cout << root->val << " ";
    inorder(root->right);
}

void postorder(Node* root) {
    if (root == nullptr) return;
    postorder(root->left);
    postorder(root->right);
    cout << root->val << " ";
}
```

三者只有 `cout` 位置不同。详见 [二叉树-前中后序遍历模板](/2026/06/04/二叉树-前中后遍历/)。

## ⚠️ 关键坑点

### 1. 建树时 index 必须是引用

```cpp
Node* buildtree(vector<char>& A, int& index)  // ✅ index 是引用
Node* buildtree(vector<char>& A, int index)   // ❌ 值传递，递归不回传
```

`index` 必须全局推进——左子树建完后的 index 位置，就是右子树开始的位置。不用引用会导致右子树读到了左子树的序列位置。

### 2. 层序遍历用队列，不是栈

队列（FIFO）保证一层走完再走下一层。用栈（LIFO）会变成深度优先，顺序完全不同。

### 3. 叶子判断条件

```cpp
if (root->left == nullptr && root->right == nullptr)
//   ^^^^^^^^^^^^^^^^^^^^^^    ^^^^^^^^^^^^^^^^^^^^^^^
//   左右都为 null 才算叶子
```

只有一个孩子为 null 不是叶子。

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Node {
    char val;
    Node *left, *right;
    Node(char x) { val = x; left = right = nullptr; }
};

Node* buildtree(vector<char>& A, int& index) {
    if (index >= A.size()) return nullptr;
    char x = A[index++];
    if (x == '0') return nullptr;
    Node* root = new Node(x);
    root->left  = buildtree(A, index);
    root->right = buildtree(A, index);
    return root;
}

void preorder(Node* root) {
    if (root == nullptr) return;
    cout << root->val << " ";
    preorder(root->left);
    preorder(root->right);
}

void inorder(Node* root) {
    if (root == nullptr) return;
    inorder(root->left);
    cout << root->val << " ";
    inorder(root->right);
}

void postorder(Node* root) {
    if (root == nullptr) return;
    postorder(root->left);
    postorder(root->right);
    cout << root->val << " ";
}

void levelorder(Node* root) {
    if (root == nullptr) return;
    queue<Node*> q;
    q.push(root);
    while (!q.empty()) {
        Node* cur = q.front(); q.pop();
        cout << cur->val << " ";
        if (cur->left)  q.push(cur->left);
        if (cur->right) q.push(cur->right);
    }
}

int countleaves(Node* root) {
    if (root == nullptr) return 0;
    if (root->left == nullptr && root->right == nullptr) return 1;
    return countleaves(root->left) + countleaves(root->right);
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    vector<char> A; char c;
    while (cin >> c) A.push_back(c);
    int index = 0;
    Node* root = buildtree(A, index);
    preorder(root);  cout << endl;
    inorder(root);   cout << endl;
    postorder(root); cout << endl;
    levelorder(root); cout << endl;  // 层序
    cout << countleaves(root) << endl;
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 先序建树 | `index` 引用全局推进，0 为空，根→左→右递归 |
| 层序遍历 | 队列 FIFO，根入队 → 出队输出 → 左右孩子入队 |
| 数叶子 | 左右都 null → 1，否则左+右 |
| 层序 vs 前序 | 层序（BFS）用队列，前序（DFS）用递归 |
