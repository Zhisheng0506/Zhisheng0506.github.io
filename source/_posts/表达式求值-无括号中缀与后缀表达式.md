## 题目描述

> 题目链接：[N诺/DreamJudge 1301](https://noobdream.com/DreamJudge/Issue/page/1301/)

计算不存在括号的中缀表达式，包含 `+` `-` `*` `/`，表达式不含空格。

```
输入: 6/2+3+3*4
输出: 18
```

<!--more-->

## 核心思路

只用一个**数值栈**，利用"遇见运算符才处理上一个运算符"的延迟策略，自然地解决了 `*` `/` 优先级高于 `+` `-` 的问题。

```
思路:
  遇见数字 → 累积
  遇见运算符（或末尾）→ 根据上一个运算符决定如何处理累积的数字
    + → 直接入栈
    - → 取负入栈
    * → 栈顶出栈 × 当前数，结果入栈
    / → 栈顶出栈 ÷ 当前数，结果入栈
  最后 → 栈中所有数字求和
```

## 算法拆解

### 变量声明

```cpp
vector<int> stk;     // 数值栈，存"待求和"的数字
int num = 0;         // 当前正在累积的数字
char sign = '+';     // 上一个运算符（初始为 +）
```

### 遍历表达式

```cpp
for (int i = 0; i < s.size(); i++) {
    if (s[i] >= '0' && s[i] <= '9') {
        num = num * 10 + (s[i] - '0');   // ① 累积数字（可能多位）
    }
    if (!(s[i] >= '0' && s[i] <= '9') || i == s.size() - 1) {
        // ② 遇到运算符或末尾 → 处理上一个运算符
        if (sign == '+')      stk.push_back(num);
        else if (sign == '-') stk.push_back(-num);
        else if (sign == '*') { int t = stk.back(); stk.pop_back(); stk.push_back(t * num); }
        else if (sign == '/') { int t = stk.back(); stk.pop_back(); stk.push_back(t / num); }

        sign = s[i];   // ③ 记录当前运算符，留给下一轮
        num = 0;       // ④ 重置 num
    }
}
```

### 触发时机

```cpp
if (!(s[i] >= '0' && s[i] <= '9') || i == s.size() - 1)
```

当前字符**不是数字**（即运算符），或走到**表达式末尾**时触发处理。末尾的特殊判断保证最后一个数字不会漏掉。

### 最终求和

```cpp
int res = 0;
while (!stk.empty()) {
    res += stk.back();
    stk.pop_back();
}
cout << res << endl;
```

## 完整执行过程

```
表达式: 6/2+3+3*4

初始: stk=[], num=0, sign='+'

i=0: '6' → num=6
i=1: '/' → 运算符!
  sign='+' → 压入 +6 → stk=[6]
  sign='/', num=0

i=2: '2' → num=2
i=3: '+' → 运算符!
  sign='/' → pop 6, 6/2=3 → stk=[3]
  sign='+', num=0

i=4: '3' → num=3
i=5: '+' → 运算符!
  sign='+' → 压入 +3 → stk=[3,3]
  sign='+', num=0

i=6: '3' → num=3
i=7: '*' → 运算符!
  sign='+' → 压入 +3 → stk=[3,3,3]
  sign='*', num=0

i=8: '4' → num=4
i=9: 末尾 → 触发!
  sign='*' → pop 3, 3*4=12 → stk=[3,3,12]

求和: 3+3+12=18
```

---

## 为什么这样能正确处理优先级

关键：`*` 和 `/` 在**遇到下一个运算符时立刻执行**，而 `+` 和 `-` 只是把数压入栈中。

```
3 + 2 * 4:

i=0: '3' → num=3
i=1: '+' → 压入 3 (sign='+')
i=2: '2' → num=2
i=3: '*' → 压入 2 (sign='+'), 还没轮到 *
i=4: '4' → num=4
末尾 → sign='*', pop 2, 2*4=8, 压入 8
栈: [3, 8] → 3+8=11 ✓
```

`*` 在遇到下一个运算符（或末尾）时才执行——此时 `num` 已经是完整的第二个因数了。

## 拓展：后缀表达式求值

后缀表达式（逆波兰表达式）运算符在操作数之后，**天然无歧义、不需要括号、不需要优先级**。求值更简单：

```cpp
int evalRPN(vector<string>& tokens) {
    stack<int> stk;
    for (auto& t : tokens) {
        if (t == "+" || t == "-" || t == "*" || t == "/") {
            int b = stk.top(); stk.pop();
            int a = stk.top(); stk.pop();
            if (t == "+") stk.push(a + b);
            if (t == "-") stk.push(a - b);
            if (t == "*") stk.push(a * b);
            if (t == "/") stk.push(a / b);
        } else {
            stk.push(stoi(t));
        }
    }
    return stk.top();
}
```

```
后缀: ["6","2","/","3","+","3","4","*","+"]

6 2 /  → 6/2=3
3 3 +  → 3+3=6
3 4 *  → 3*4=12
6 12 + → 6+12=18
结果: 18
```

### 中缀 vs 后缀

| | 中缀 | 后缀 |
|------|------|------|
| 形式 | `3 + 4 * 2` | `3 4 2 * +` |
| 括号 | 需要 | 不需要 |
| 优先级 | 需要处理 | 不需要 |
| 求值算法 | 数值栈 + 延迟运算 | 数值栈，遇运算符就出栈两个 |
| 复杂度 | O(N) | O(N) |

后缀表达式只需要一个栈、一个规则——"遇数字入栈，遇运算符弹出两个计算后入栈"。没有优先级判断，没有延迟处理，比中缀简单得多。

## ⚠️ 关键坑点

### 1. 最后一位必须触发

```cpp
if (!(s[i] >= '0' && s[i] <= '9') || i == s.size() - 1)
//                                   ^^^^^^^^^^^^^^^^ 末尾！
```

漏了这个条件，最后一个数字永远不会被处理。

### 2. 用上一个运算符，不是当前

```cpp
sign = s[i];  // 记录当前运算符，留给**下一轮**用
```

每次处理的是 **上一个** `sign`，当前这个存起来等下一轮。

### 3. 除法是 `top / num`，不是 `num / top`

```cpp
stk.push_back(top / num);  // ✅ 栈顶 ÷ 当前数
```

栈顶是先出现的操作数（被除数），`num` 是后出现的（除数）。

### 4. 带括号的表达式

本文算法不含括号。带括号的中缀需要**双栈**（操作数栈 + 运算符栈）或先转后缀再求值。

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;
    while (cin >> s) {
        vector<int> stk;
        int num = 0;
        char sign = '+';

        for (int i = 0; i < s.size(); i++) {
            if (s[i] >= '0' && s[i] <= '9') {
                num = num * 10 + (s[i] - '0');
            }
            if (!(s[i] >= '0' && s[i] <= '9') || i == s.size() - 1) {
                if (sign == '+') stk.push_back(num);
                else if (sign == '-') stk.push_back(-num);
                else if (sign == '*') {
                    int t = stk.back(); stk.pop_back();
                    stk.push_back(t * num);
                } else if (sign == '/') {
                    int t = stk.back(); stk.pop_back();
                    stk.push_back(t / num);
                }
                sign = s[i];
                num = 0;
            }
        }

        int res = 0;
        for (int x : stk) res += x;
        cout << res << endl;
    }
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 核心技巧 | 只有一个数值栈，利用"延迟运算"处理优先级 |
| + - | 直接入栈（或取负入栈） |
| * / | 立刻出栈运算，结果入栈 |
| 触发时机 | 遇到运算符 或 走到末尾 |
| 后缀求值 | 遇数字入栈，遇运算符弹二算一 |
| 无括号 | 本文算法适用于无括号中缀，有括号需要双栈 |
