二叉树-前中后序遍历模板

Posted on 2026-06-04

题目描述

题目链接：N诺/DreamJudge 1332

输入 n 个整数，建立二叉排序树（BST），依次输出前序、中序、后序遍历结果。

这题就是模板题——背熟模板、理解遍历顺序，直接默写。

树节点定义

l 是左子树指针，r 是右子树指针。初始都为 nullptr。

BST 插入

Node* &root 是指针的引用——加了 & 才能在递归中真正修改指针的值（把 nullptr 变成 new Node(x)）。不加 & 只改局部副本，树建不起来。

三种遍历

三种遍历的唯一区别是访问根节点的时机：

前序：根 → 左 → 右

中序：左 → 根 → 右

BST 的中序遍历结果一定是从小到大有序的。 这是机试中常用的验证技巧——如果中序不是升序，说明树建错了。

后序：左 → 右 → 根

记忆口诀：

  • 前序 = 根左右
  • 中序 = 左根右
  • 后序 = 左右根

⚠️ 关键坑点

1. 指针引用 Node* &root

不加 &，root = new Node(x) 只改局部副本，调用者的指针还是 nullptr。树建完还是空的，遍历什么都不输出。

2. BST 相等值要忽略

漏了 else if 中的 > 判断，相等值可能死循环递归。

3. 遍历判空必须在第一行

如果先访问 root->val 再判空，走到叶子的子节点时直接段错误。

4. 函数名和调用要一致

三个位置要写同一个函数名。最常见的错误：写 preorder 递归内部又写了 inorder。

总结

要点 说明
前序 根左右 — cout → l → r
中序 左根右 — l → cout → r（BST 中序 = 有序）
后序 左右根 — l → r → cout
模板差异 只有 cout 位置不同
插入 Node* &root（指针引用），小于往左大于往右
判空 遍历函数第一行必须 if(root==nullptr) return
