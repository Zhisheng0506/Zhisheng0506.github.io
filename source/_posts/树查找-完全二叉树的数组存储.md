## 题目描述

> 题目链接：[N诺/DreamJudge 1330](https://noobdream.com/DreamJudge/Issue/page/1330/)

有一棵完全二叉树，输出某一深度的所有节点，有则输出，无则输出 `EMPTY`。

```
输入:
4
1 2 3 4
2

输出:
2 3
```

```
树结构:
        1          ← 第 1 层
       / \
      2   3        ← 第 2 层
     /
    4              ← 第 3 层

d=2 → 输出 2 3
```

`1 ≤ n ≤ 1000`。

<!--more-->

## 核心思路

**完全二叉树最好的存储方式是数组**，下标从 1 开始：

```
数组下标:  1   2   3   4
节点值:    1   2   3   4

下标 i 的左孩子: 2i
下标 i 的右孩子: 2i + 1
下标 i 的父节点: i / 2
```

### 第 d 层的下标范围

```
第 1 层: [1, 1]       = [2^0, 2^1-1]
第 2 层: [2, 3]       = [2^1, 2^2-1]
第 3 层: [4, 7]       = [2^2, 2^3-1]
第 d 层: [2^(d-1), 2^d - 1]
```

```cpp
int start = pow(2, d - 1);               // 第 d 层第一个节点
int end   = min(n, (int)pow(2, d) - 1);  // 第 d 层最后一个，不超 n
```

---

## ⚠️ 为什么 `pow` 必须强转

```cpp
// ❌ 编译报错
int end = min(n, pow(2, d) - 1);

// ✅ 必须强转
int end = min(n, (int)pow(2, d) - 1);
```

`pow` 返回的是 `double`，`pow(2, d) - 1` 也是 `double`。`min` 要求两个参数类型**完全一致**，`n` 是 `int`，`pow(...)` 是 `double` → 编译报错。

```cpp
min(3, 5);        // ✅ 都是 int
min(3.0, 5.0);    // ✅ 都是 double
min(3, 5.0);      // ❌ int vs double → 编译错误
min(3, (int)5.0); // ✅ 强转后一致
```

**任何用到 `pow` 并参与 `min`/`max` 的地方，一律 `(int)` 强转。**

---

## 完全二叉树：数组 vs 链表

| | 数组存储 | 链表存储 |
|------|------|------|
| 找子节点 | `A[2i]` `A[2i+1]` | `root->l` `root->r` |
| 找父节点 | `i/2` | 需要额外指针 |
| 某层范围 | `[2^(d-1), 2^d-1]`，O(1) | 需要层序遍历 |
| 适用 | 完全二叉树 | 任意二叉树 |

对于完全二叉树，数组在时间、空间、代码量上全面占优。

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    while (cin >> n) {
        vector<int> A(n + 1);
        for (int i = 1; i <= n; i++) {
            cin >> A[i];
        }
        int d; cin >> d;

        int start = pow(2, d - 1);
        int end = min(n, (int)pow(2, d) - 1);

        if (start > n) {
            cout << "EMPTY" << endl;
        } else {
            for (int i = start; i <= end; i++) {
                cout << A[i];
                if (i < end) cout << " ";
            }
            cout << endl;
        }
    }
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 完全二叉树存储 | 数组，下标从 1 开始 |
| 左/右孩子 | `2i` / `2i + 1` |
| 第 d 层范围 | `[2^(d-1), min(n, 2^d - 1)]` |
| **pow 强转** | `(int)pow(2, d)`，否则 min 参数类型不匹配 |
| 无该层 | `start > n` → EMPTY |
