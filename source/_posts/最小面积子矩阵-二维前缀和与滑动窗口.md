## 题目描述

> 题目链接：[N诺/DreamJudge 1285](https://noobdream.com/DreamJudge/Issue/page/1285/)

在 N×M 矩阵中，找出所有元素和不小于 K 的**面积最小的子矩阵**。若不存在则输出 -1。

暴力枚举子矩阵是 $O(N^2 M^2)$，四个循环必超时。正确做法是**枚举行范围 + 列前缀和压缩 + 一维滑动窗口**，复杂度 $O(N^2 M)$。

<!--more-->

## 整体思路

分三步：

1. **枚举行的上下边界**（i, j），确定子矩阵的高度
2. **列前缀和压缩**：把 i~j 行之间每一列压成一个数，变成一维数组
3. **一维滑动窗口**：在压缩后的一维数组上，找满足和 ≥ K 的最短子数组

```
原始 2D 矩阵               压缩为 1D 数组（每列压成一个数）
┌───────────┐             ┌───────┐
│ 1  2  3  4│             │ s1│s2│s3│s4│
│ 5  6  7  8│  →  行 i~j 列和拆成 (s1, s2, s3, s4)
│ 9 10 11 12│
│13 14 15 16│             然后滑动窗口找和 ≥ K 的最短子数组
└───────────┘
```

## 第一步：列前缀和

对每一列单独做前缀和，`colsum[i][j]` 表示第 j 列从第 1 行到第 i 行的和：

```cpp
vector<vector<int>> mtx(n + 1, vector<int>(m + 1));
vector<vector<int>> colsum(n + 1, vector<int>(m + 1));

for (int i = 1; i <= n; i++) {
    for (int j = 1; j <= m; j++) {
        cin >> mtx[i][j];
        colsum[i][j] = colsum[i - 1][j] + mtx[i][j];
    }
}
```

### ⚠️ 为什么要从 1 开始

前缀和的核心操作是 `colsum[i][j] = colsum[i-1][j] + mtx[i][j]`。如果 i=0，`colsum[-1][j]` 就越界了。

从 1 开始的好处：
- `colsum[0][j]` 天然为 0，无需特判
- 任意区间和 = `colsum[j][k] - colsum[i-1][k]`，公式统一

**机试中只要涉及前缀和，一律从下标 1 开始，省去无数边界特判。**

## 第二步：枚举行范围，压成一维

```cpp
for (int i = 1; i <= n; i++) {           // 上边界
    for (int j = i; j <= n; j++) {        // 下边界
        int height = j - i + 1;           // 当前子矩阵高度

        // 把 i~j 行的每一列压成一个数
        vector<int> colsumInrange(m + 1);
        for (int k = 1; k <= m; k++) {
            colsumInrange[k] = colsum[j][k] - colsum[i - 1][k];
        }

        // 现在问题变成：在 colsumInrange 这个一维数组中，
        // 找满足和 ≥ K 的最短子数组
    }
}
```

**这是整个算法最关键的一步**：用前缀和的差，把二维子矩阵的行方向压缩掉，每一列变成一个数，问题降维。

## 第三步：一维滑动窗口

现在 `colsumInrange` 是一个长度为 m 的数组，要求其中**和 ≥ K 的最短连续子数组**。

```cpp
int currentsum = 0, left = 1;
for (int right = 1; right <= m; right++) {
    currentsum += colsumInrange[right];   // 右指针扩展窗口

    while (currentsum >= k && left <= right) {
        int width = right - left + 1;
        int area = height * width;
        minarea = min(minarea, area);

        currentsum -= colsumInrange[left];
        left++;
    }
}
```

### 滑动窗口模板复盘

```
模板：
for (right = 0; right < n; right++) {
    加入 right 元素
    while (窗口满足条件) {
        更新答案
        移出 left 元素
        left++
    }
}
```

关键点：
- **右指针不断扩展**，每次循环前进一格
- **左指针只在满足条件时收缩**，`while` 保证收缩到底
- 每次满足条件时更新答案（求最短用 `min`）

本题中窗口内是 `colsumInrange` 连续一段，窗口和 = 子矩阵所有元素的和。行已固定，列方向用滑动窗口找最短满足条件的宽度。

## ⚠️ INT_MAX 与 INT_MIN 的用法

### INT_MAX — 求最小值时的哨兵

```cpp
int minarea = INT_MAX;  // 初始化为最大值

// 滑动窗口过程中更新
minarea = min(minarea, area);

// 最后判断
if (minarea == INT_MAX) {
    cout << "-1" << endl;  // 从未更新过 → 无解
}
```

`INT_MAX`（2147483647）作为"最小值"的初始哨兵值，天然保证任何有效面积都比它小。无解时它保持原样，方便判断。

### INT_MIN — 求最大值时的哨兵

对称地，如果要找**最大值**，初始化为 `INT_MIN`（-2147483648）：

```cpp
int maxval = INT_MIN;

for (...) {
    maxval = max(maxval, current);
}

if (maxval == INT_MIN) {
    // 从未更新过，无解
}
```

典型场景：找数组中最大的子数组和、最大的满足某条件的值。

### 什么时候用它们

| 场景 | 初始值 | 判断无解 |
|------|--------|----------|
| 求**最小**值 | `INT_MAX` | `== INT_MAX` |
| 求**最大**值 | `INT_MIN` | `== INT_MIN` |

核心原则：**用和求解方向相反的极值做哨兵**——求最小就用最大初始化，求最大就用最小初始化。

### ⚠️ 容易写错

- 求最小值却用 0 初始化 → 负数答案会被漏掉
- 求最大值却用 -1 初始化 → 元素全是负数时错判
- 无解判断如果用 `> 某个数` 而不是 `== 极值` → 可能漏判

一句话：**用 `INT_MAX` / `INT_MIN` 永远不会和真实值混淆。**

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m, k;
    while (cin >> n >> m >> k) {
        vector<vector<int>> mtx(n + 1, vector<int>(m + 1));
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {
                cin >> mtx[i][j];
            }
        }

        vector<vector<int>> colsum(n + 1, vector<int>(m + 1));
        for (int i = 1; i <= n; i++) {
            for (int j = 1; j <= m; j++) {
                colsum[i][j] = colsum[i - 1][j] + mtx[i][j];
            }
        }

        int minarea = INT_MAX;

        for (int i = 1; i <= n; i++) {
            for (int j = i; j <= n; j++) {
                int height = j - i + 1;

                vector<int> colsumInrange(m + 1);
                for (int k = 1; k <= m; k++) {
                    colsumInrange[k] = colsum[j][k] - colsum[i - 1][k];
                }

                int currentsum = 0, left = 1;
                for (int right = 1; right <= m; right++) {
                    currentsum += colsumInrange[right];

                    while (currentsum >= k && left <= right) {
                        int width = right - left + 1;
                        int area = height * width;
                        minarea = min(minarea, area);

                        currentsum -= colsumInrange[left];
                        left++;
                    }
                }
            }
        }

        if (minarea == INT_MAX) {
            cout << "-1" << endl;
        } else {
            cout << minarea << endl;
        }
    }
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 前缀和下标 | 从 1 开始，省去边界特判 |
| 列前缀和 | `colsum[i][j] = colsum[i-1][j] + mtx[i][j]` |
| 压缩公式 | `colsumInrange[k] = colsum[j][k] - colsum[i-1][k]` |
| 滑动窗口模板 | 右指针扩展 + while 满足时收缩左指针 |
| 复杂度 | $O(N^2 M)$，枚举行 $O(N^2)$ × 滑动窗口 $O(M)$ |
| min 哨兵 | `INT_MAX` 初始化，无解时保持原值 |
