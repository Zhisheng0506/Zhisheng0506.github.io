## 题目描述

> 题目链接：[N诺/DreamJudge 1287](https://noobdream.com/DreamJudge/Issue/page/1287/)

凯撒密码：输入一个由字母组成的字符串，每个字符向后偏移 2 位（a→c, Z→B）。输出每个偏移后字符的**奇校验码**及其对应的**十进制数**。

奇校验规则：如果字符二进制中 1 的个数为**偶数**，将最高位置为 1（确保 1 的总数为奇数）。

```
输入: aD
输出:
原文：aD
密文：cF
c  11100011  227
F  01000110  70
```

<!--more-->

## 核心思路

这道题分为四步：

```
原文 → 密文(+2) → 十进制转二进制 → 奇校验 → 二进制转回十进制
```

其中两步进制转换是本篇的重点——用位运算替代 %2/÷2 和逐位加权。这是对 [位运算技巧-以压缩日期为例](/2026/05/26/位运算技巧-以压缩日期为例/) 的知识复习和实战应用。

---

## 第一步：凯撒偏移

```cpp
for (int i = 0; i < s.size(); i++) {
    s[i] = s[i] + 2;  // ASCII 码 +2
}
```

'a'(97) + 2 = 'c'(99)。题目不要求循环回绕（Z→B），直接 +2 即可。

---

## 第二步：十进制 → 二进制（位运算）

### 旧方法 vs 位运算

```cpp
// ❌ 旧方法：%2 /2 + reverse
string s = "";
while (n) {
    s += (n % 2) + '0';
    n /= 2;
}
reverse(s.begin(), s.end());
```

```cpp
// ✅ 位运算：(n >> i) & 1，逐位提取
int n = 227;
for (int i = 31; i >= 0; i--) {
    cout << ((n >> i) & 1);
}
```

`(n >> i) & 1` 就是上一篇位运算博客中位解包的核心模式——现在它被用来逐位提取二进制。

### 原理

```
n = 227  →  00000000 00000000 00000000 11100011

(n >> 7) & 1:
  227 >> 7  = 00000000 00000000 00000000 00000001    (第7位是1)
  & 1       = 00000000 00000000 00000000 00000001  = 1

(n >> 6) & 1:
  227 >> 6  = 00000000 00000000 00000000 00000000    (第6位是0)
  & 1       = 00000000 00000000 00000000 00000000  = 0
```

两步：`>> i` 把第 i 位移到最低位，`& 1` 只保留那一位。

### 去掉前导零

32 位全输出会有大量前导 0。去掉它们（至少保留一位）：

```cpp
string x = "";
for (int i = 31; i >= 0; i--) {
    x += char(((n >> i) & 1) + '0');
}
while (x.size() > 1 && x[0] == '0') {
    x.erase(0, 1);
}
```

`x.size() > 1` 保证即使原值是 0，也能保留一个 `"0"`。

## 第三步：奇校验

```cpp
// 统计 1 的个数
int cnt = 0;
for (int i = 0; i < x.size(); i++) {
    if (x[i] == '1') cnt++;
}

// 奇校验规则
if (cnt % 2 == 0) {
    x = "1" + x;   // 偶数个1 → 最高位补1，变成奇数个
} else {
    x = "0" + x;   // 奇数个1 → 补0，个数不变
}
```

```
'c' = 99
  二进制: 1100011
  1的个数: 1+1+0+0+0+1+1 = 4（偶数）
  → 奇校验: 1 + 1100011 = 11100011 = 227 ✓

'F' = 70
  二进制: 1000110
  1的个数: 1+0+0+0+1+1+0 = 3（奇数）
  → 奇校验: 0 + 1000110 = 01000110 = 70 ✓
```

| 1 的个数 | 操作 | 效果 |
|----------|------|------|
| 奇数 | 最高位加 `0` | 1 的个数不变 |
| 偶数 | 最高位加 `1` | 多一个 1，变成奇数 |

## 第四步：二进制 → 十进制（位运算）

### 旧方法 vs 位运算

```cpp
// ❌ 逐位加权: ans = ans * 2 + bit
int ans = 0;
for (int i = 0; i < x.size(); i++) {
    ans = ans * 2 + (x[i] - '0');
}
```

```cpp
// ✅ 位运算: ans = (ans << 1) | bit
int ans = 0;
for (int i = 0; i < x.size(); i++) {
    ans = (ans << 1) | (x[i] - '0');
}
```

`(ans << 1) | bit` 正是上一篇博客中位打包的核心模式——现在被用来从二进制位串构建十进制数。

### 原理（以 `x = "11100011"` 为例）

```
初始:  ans = 0 (二进制 0)

i=0: bit=1
  ans << 1  →  0          (0 左移一位还是 0)
  | 1       →  1

i=1: bit=1
  ans << 1  →  10         (1 左移一位 = 2)
  | 1       →  11         (2|1 = 3)

i=2: bit=1
  ans << 1  →  110        (3 左移一位 = 6)
  | 1       →  111        (6|1 = 7)

i=3: bit=0
  ans << 1  →  1110       (7 左移一位 = 14)
  | 0       →  1110       (14|0 = 14)

...

最终: ans = 11100011₂ = 227
```

**一句话**：左移腾出最低位，`|` 把新位放进去。`ans = (ans << 1) | bit` 等价于 `ans = ans * 2 + bit`，但位运算写法更"算法竞赛"。

## 和上一篇位运算博客的呼应

两个核心模式在两篇博客中反复出现：

| 模式 | 上一篇（压缩日期） | 本篇（进制转换） |
|------|------|------|
| `<<` 和 `\|` 拼接 | `(year2<<9) \| (day<<4) \| month` 位打包 | `(ans<<1) \| bit` 拼接二进制位 |
| `>>` 和 `&` 提取 | `(ans>>9) & 0x7F` 解包字段 | `(n>>i) & 1` 逐位提取二进制 |

同一套位运算技能，在不同场景下的两种运用。懂了这两个模式，机试中位运算相关题目基本就覆盖了。

## ⚠️ 关键坑点

### 1. 循环从 31 开始（不是 7）

```cpp
for (int i = 31; i >= 0; i--)  // ✅ int 是 32 位
// 不是:
for (int i = 7; i >= 0; i--)   // ❌ 赋值给 int n 后变成了 32 位
```

`s[i]` 虽然是 char（8 位），但赋值给了 `int n`，所以是 32 位，从 31 开始遍历。

### 2. 去前导零要至少保留一位

```cpp
while (x.size() > 1 && x[0] == '0') {
    x.erase(0, 1);
}
```

`x.size() > 1` 保证值为 0 时至少保留 `"0"`。

### 3. `& 1` 的结果转字符要 + '0'

```cpp
x += char(((n >> i) & 1) + '0');
//        ^^^^^^^^^^^^    ^^^^^
//        整数 0 或 1      转成字符
```

`((n>>i)&1)` 是整数 0 或 1。`+ '0'`（即 +48）变成字符 `'0'` 或 `'1'`。

### 4. 奇校验加在字符串**前面**（最高位）

```cpp
x = "1" + x;   // ✅ 前面 = 最高位
// 不是:
x = x + "1";   // ❌ 后面 = 最低位，语义错误
```

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    string s;
    cin >> s;
    cout << "原文：" << s << endl;

    string ss = "";
    for (int i = 0; i < s.size(); i++) {
        ss += s[i] + 2;
    }
    cout << "密文：" << ss << endl;

    for (int i = 0; i < s.size(); i++) {
        s[i] = s[i] + 2;
        cout << s[i] << "  ";

        // 十进制 → 二进制（位运算）
        int n = (int)s[i];
        string x = "";
        for (int i = 31; i >= 0; i--) {
            x += char(((n >> i) & 1) + '0');
        }
        while (x.size() > 1 && x[0] == '0') {
            x.erase(0, 1);
        }

        // 奇校验
        int cnt = 0;
        for (int i = 0; i < x.size(); i++) {
            if (x[i] == '1') cnt++;
        }
        if (cnt % 2 == 0) {
            x = "1" + x;
        } else {
            x = "0" + x;
        }

        cout << x << "  ";

        // 二进制 → 十进制（位运算）
        int ans = 0;
        for (int i = 0; i < x.size(); i++) {
            ans = (ans << 1) | (x[i] - '0');
        }
        cout << ans << endl;
    }
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 十进制 → 二进制 | `(n >> i) & 1`，从高位到低位逐位提取 |
| 二进制 → 十进制 | `(ans << 1) \| bit`，左移腾位 + 放入新位 |
| 去前导零 | `while (size>1 && x[0]=='0') erase(0,1)` |
| 奇校验 | 偶数个1 → 最高位补1；奇数个1 → 补0 |
| 循环范围 | int 从 31 开始（32 位），不是 7 |
| 与上一篇呼应 | `<<` `\|` 用于拼接，`>>` `&` 用于提取，同模式 |
