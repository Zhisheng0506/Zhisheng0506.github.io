## 题目描述

> 题目链接：[N诺/DreamJudge 1300](https://noobdream.com/DreamJudge/Issue/page/1300/)

给定二叉树的前序遍历和中序遍历，求后序遍历。

```
输入:
ABC
BAC
FDXEAG
XDEFAG

输出:
BCA
XEDGAF
```

`n ≤ 26`，结点以大写字命名。

<!--more-->

## 核心思路

前序确定根，中序划分左右子树。

```
前序: A B C
中序: B A C

1. 前序第一个 'A' 是根
2. 在中序找到 'A' → 左边 "B" 是左子树，右边 "C" 是右子树
3. 递归处理左子树和右子树
4. 后序 = 左后序 + 右后序 + 根
```

---

## 算法拆解

```cpp
string getPostOrder(string pre, string in) {
    if (pre.empty()) return "";

    char root = pre[0];                          // ① 前序第一个 = 根
    int rootIndex = in.find(root);               // ② 根在中序的位置

    // ③ 根据根的位置，切分中序
    string leftIn  = in.substr(0, rootIndex);
    string rightIn = in.substr(rootIndex + 1);

    // ④ 根据左子树长度，切分前序
    string leftPre  = pre.substr(1, leftIn.size());
    string rightPre = pre.substr(leftIn.size() + 1);

    // ⑤ 后序 = 左后序 + 右后序 + 根
    return getPostOrder(leftPre, leftIn)
         + getPostOrder(rightPre, rightIn)
         + root;
}
```

## 完整执行过程

```
前序: F D X E A G
中序: X D E F A G

根 = 'F'
中序中 'F' 位置 = 3
  leftIn  = "XDE"  (中序 0~2)
  rightIn = "AG"   (中序 4~5)

左子树长度 = 3
  leftPre  = "DXE"  (前序 1~3)
  rightPre = "AG"   (前序 4~5)

递归左:
  前序 "DXE", 中序 "XDE"
  根 = 'D', 位置 = 1
  leftIn = "X", leftPre = "X"
  rightIn = "E", rightPre = "E"
  
  递归 leftPre="X", leftIn="X":
    根='X', 位置=0 → 左右都空 → 返回 "X"
  递归 rightPre="E", rightIn="E":
    根='E', 位置=0 → 左右都空 → 返回 "E"
  返回 "X" + "E" + "D" = "XED"

递归右:
  前序 "AG", 中序 "AG"
  根 = 'A', 位置 = 0
  leftIn="" rightIn="G", leftPre="" rightPre="G"
  递归右: 前序 "G", 中序 "G" → "G"
  返回 "" + "G" + "A" = "GA"

最终: "XED" + "GA" + "F" = "XEDGAF"
```

## 为什么前序+中序能唯一确定一棵树

- **前序第一个确定根节点**
- **中序中根的位置确定左右子树的节点集合**
- 左右子树又是同样的结构 → 递归

前序+后序**不能**唯一确定（缺少左右子树的分界信息）。但前序+中序、中序+后序都可以。

---

## ⚠️ 关键坑点

### 1. 前序切分时的偏移

```cpp
string leftPre  = pre.substr(1, leftIn.size());    // 从1开始，长度=左子树节点数
string rightPre = pre.substr(leftIn.size() + 1);   // 跳过根+左子树
```

前序的布局是 `[根][左子树][右子树]`。左前序跳过根（下标 1），右前序跳过根+左子树。

### 2. `find` 返回值是 size_t

```cpp
int rootIndex = in.find(root);  // ✅ 存 int
size_t rootIndex = in.find(root); // 也可以，但和 substr 混用要注意符号
```

用 `int` 更安全，避免 `size_t` 和负数比较时的隐式转换问题。

### 3. 空串终止条件

```cpp
if (pre.empty()) return "";  // 或 pre.size() == 0
```

叶子节点递归下去，左右子树为空，返回空串。

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

string getPostOrder(string pre, string in) {
    if (pre.empty()) return "";

    char root = pre[0];
    int rootIndex = in.find(root);

    string leftIn  = in.substr(0, rootIndex);
    string rightIn = in.substr(rootIndex + 1);

    string leftPre  = pre.substr(1, leftIn.size());
    string rightPre = pre.substr(leftIn.size() + 1);

    return getPostOrder(leftPre, leftIn)
         + getPostOrder(rightPre, rightIn)
         + root;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    string pre, in;
    while (cin >> pre >> in) {
        cout << getPostOrder(pre, in) << endl;
    }
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 核心 | 前序首字符 = 根，中序根位置分左右 |
| 前序布局 | `[根][左子树][右子树]` |
| 切分 | 左 pre 从 1 开始，右 pre 跳过根+左子树长度 |
| 递归公式 | 后序 = 左后序 + 右后序 + 根 |
| 终止条件 | `pre.empty()` → `""` |
