## 题目描述

> 题目链接：[N诺/DreamJudge 1353](https://noobdream.com/DreamJudge/Issue/page/1353/)

梅森数：形如 $2^p - 1$ 的数，其中 p 是素数。如果梅森数本身也是素数，就叫梅森素数。输入正整数 n，输出所有 $≤ n$ 的梅森素数。

```
输入: 100
输出:
M(2)=3
M(3)=7
M(5)=31
M(7)=127
```

<!--more-->

## 核心思路

最自然的想法是遍历 [2, n]，对每个数判断是否是 $2^p-1$ 的形式、p 是否素数、自身是否素数。但这样复杂度过高。

**反过来想**：满足 $2^p-1 ≤ n$ 的 p 很少——因为 $2^p$ 以指数增长！直接枚举 p，判断 $2^p-1$ 是否 ≤ n 且是素数。

## 快速幂

### 为什么需要快速幂

$2^p$ 可以直接用 `1LL << p`（p 不大时），但快速幂是通用模板，适用于任意底数的幂运算。这里以通用写法为准。

### 原理：二分拆分指数

```
计算 a^13:

13 的二进制: 1101
a^13 = a^8 × a^4 × a^1

拆分: 13 = 8 + 4 + 1, 对应的二进制位分别是 1,1,0,1

遍历 13 的二进制位:
  第0位=1 → res *= a^(2^0) = a^1
  第1位=0 → 跳过
  第2位=1 → res *= a^(2^2) = a^4
  第3位=1 → res *= a^(2^3) = a^8

最终: res = a^1 × a^4 × a^8 = a^13
```

每一步指数翻倍（`a *= a`），指数向右移一位（`b >>= 1`）。

### 模板

```cpp
ll qpow(ll a, ll b) {
    ll res = 1;
    while (b > 0) {
        if (b & 1) res *= a;   // 当前二进制位是 1 → 乘上 a 的当前幂
        a *= a;                 // a 不断平方: a^1, a^2, a^4, a^8, ...
        b >>= 1;                // 指数右移一位
    }
    return res;
}
```

### 执行过程（a=2, b=13）

```
b=13 (1101):
  b&1=1 → res *= 2    → res=2
  a *= a → a=4
  b>>=1 → b=6

b=6 (0110):
  b&1=0 → 跳过
  a *= a → a=16
  b>>=1 → b=3

b=3 (0011):
  b&1=1 → res *= 16   → res=32
  a *= a → a=256
  b>>=1 → b=1

b=1 (0001):
  b&1=1 → res *= 256  → res=8192 = 2^13 ✓
  a *= a → a=65536
  b>>=1 → b=0 → 退出

返回 8192
```

复杂度 O(log b)，比朴素 O(b) 快指数级。

## 带取模的快速幂

实际机试中，幂运算结果往往巨大，题目会要求对某个数取模。此时快速幂需要带上 `% mod`：

```cpp
const int mod = 233333;

ll qpow(ll x, ll n) {
    ll res = 1;
    x %= mod;                     // 底数先取模
    while (n > 0) {
        if (n & 1) {
            res = (res * x) % mod; // 乘法后立刻取模
        }
        x = (x * x) % mod;         // 平方后立刻取模
        n >>= 1;
    }
    return res;
}
```

### 和不取模版的区别

```cpp
// 不取模版
res *= a;
a *= a;

// 取模版
res = (res * x) % mod;
x = (x * x) % mod;
```

每一步乘法之后立刻 `% mod`，防止中间结果溢出。模运算的性质保证了 `(a * b) % mod = ((a % mod) * (b % mod)) % mod`，所以提前取模不影响最终结果。

### ⚠️ `x %= mod` 在函数开头

```cpp
x %= mod;  // ✅ 底数先取模
```

如果 x 本身就很大（比如 x = 10^9），不先取模的话 `x * x` 直接溢出 long long。先 `x %= mod` 把底数压到 mod 以内，后续所有乘法都在安全范围内。

### 完整示例

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;
const int mod = 233333;

ll qpow(ll x, ll n) {
    ll res = 1;
    x %= mod;
    while (n > 0) {
        if (n & 1) { res = (res * x) % mod; }
        x = (x * x) % mod;
        n >>= 1;
    }
    return res;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int x, n;
    while (cin >> x >> n) {
        cout << qpow(x, n) << endl;
    }
}
```

## 为什么 p 只枚举到 63

```cpp
for (ll p = 2; p <= 63; p++) {
    ll mst = qpow(2, p) - 1;
    if (mst > n) break;    // 超了就直接停
}
```

`2^63 ≈ 9.22 × 10^18`，已超过 `long long` 范围（`2^63 - 1`）。p=63 是 `long long` 能存下的最大 2 的幂。题目输入 n 在 `long long` 范围内，p 枚举到 63 就够。

## 素数判断

```cpp
bool isPrime(int x) {
    if (x < 2) return false;
    for (int i = 2; i * i <= x; i++) {
        if (x % i == 0) return false;
    }
    return true;
}
```

检查到 √x 即可——如果 x 有大于 √x 的因子，必有小于 √x 的因子与之配对。复杂度 O(√x)。

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;
using ll = long long;

bool isPrime(int x) {
    if (x < 2) return false;
    for (int i = 2; i * i <= x; i++) {
        if (x % i == 0) return false;
    }
    return true;
}

ll qpow(ll a, ll b) {
    ll res = 1;
    while (b > 0) {
        if (b & 1) res *= a;
        a *= a;
        b >>= 1;
    }
    return res;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    ll n; cin >> n;
    for (ll p = 2; p <= 63; p++) {
        if (isPrime(p)) {
            ll mst = qpow(2, p) - 1;
            if (mst > n) break;
            if (isPrime(mst)) {
                printf("M(%lld)=%lld\n", p, mst);
            }
        }
    }
}
```

## ⚠️ 关键坑点

### 1. 循环方向别搞反

```cpp
// ✅ 枚举 p（指数），数量极少
for (p = 2; p <= 63; p++) { ... }

// ❌ 枚举 [1, n] 再判断是否是 2^p-1 — n 可能极大，必然超时
for (int i = 1; i <= n; i++) { ... }
```

这题的思维转换：**n 很大，但满足条件的 p 很少**——不要从 n 往下找，从 p 往上生成。

### 2. 超出 n 立即 break

```cpp
if (mst > n) break;  // 不用 continue，后面只会更大
```

$2^p-1$ 随 p 单调递增，一个超了后面全超，直接 break。

### 3. 快速幂的 `a *= a` 是核心

```cpp
a *= a;   // a 不断翻倍：a → a^2 → a^4 → a^8 → ...
b >>= 1;  // 检查每一位为 1 时把当前 a 乘到结果
```

## 总结

| 要点 | 说明 |
|------|------|
| 快速幂 | `while(b)`，`b&1` 时乘入结果，`a*=a`，`b>>=1` |
| 复杂度 | O(log b)，比朴素快指数倍 |
| 枚举方向 | 枚举 p（指数），不是枚举 n 范围内的数 |
| p 上限 | 63，因为 `2^63` 已到 long long 上限 |
| break | `mst > n` 直接 break，单调递增 |
