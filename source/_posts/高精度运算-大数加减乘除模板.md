## 概述

当数字远超 `long long` 范围（比如几百位甚至上千位），就不能用基本类型运算了。高精度（大数运算）用**数组或 vector 逐位模拟**手算过程。

### 存储方式

**倒序存储**：数组下标 0 存个位、下标 1 存十位……因为进位/借位操作都是从低位往高位走，倒序最自然。

```
数字 12345 存为:
vector: [5, 4, 3, 2, 1]
下标:     0  1  2  3  4
          个  十  百  千  万
```

输入时把字符串反转，输出时再从后往前打印。

## 高精度加法

```cpp
vector<int> add(const vector<int>& a, const vector<int>& b) {
    vector<int> c;
    int carry = 0;
    for (int i = 0; i < a.size() || i < b.size(); i++) {
        if (i < a.size()) carry += a[i];    // 加上 a 的当前位
        if (i < b.size()) carry += b[i];    // 加上 b 的当前位
        c.push_back(carry % 10);            // 当前位结果
        carry /= 10;                        // 进位
    }
    if (carry > 0) c.push_back(carry);      // 最高位有进位
    return c;
}
```

### 原理

和手算竖式加法一样：从低位到高位逐位相加，满 10 进 1。

```
    1 2 3 4
  +   5 6 7
  ---------
    1 8 0 1

倒序存为:
a = [4, 3, 2, 1]
b = [7, 6, 5]

i=0: carry=4+7=11 → c[0]=1, carry=1
i=1: carry=1+3+6=10 → c[1]=0, carry=1
i=2: carry=1+2+5=8 → c[2]=8, carry=0
i=3: carry=0+1=1 → c[3]=1
→ c=[1,0,8,1] → 1801
```

## 高精度减法

```cpp
vector<int> sub(vector<int>& a, vector<int>& b) {
    vector<int> c = a;                      // 复制 a 作为结果
    for (int i = 0; i < c.size(); i++) {
        if (i < b.size()) c[i] -= b[i];    // 减去 b 的当前位
        if (c[i] < 0) {                    // 不够减 → 借位
            c[i] += 10;
            c[i + 1] -= 1;
        }
    }
    while (c.size() > 1 && c.back() == 0)   // 去除前导零
        c.pop_back();
    return c;
}
```

### 原理

和手算减法一样：从低到高逐位相减，不够借 1 当 10。

```
    1 2 0 0
  -     5 6
  ---------
    1 1 4 4

a=[0,0,2,1], b=[6,5]

i=0: 0-6=-6<0 → 0+10=10, 借位 → c[0]=4, c[1]=-1
i=1: -1-5=-6 → -6+10=4, 借位 → c[1]=4, c[2]=1
i=2: 1-0=1 → c[2]=1
i=3: 1-0=1 → c[3]=1
→ c=[4,4,1,1] → 1144
```

### ⚠️ 假设 a ≥ b

这个模板假设被减数 ≥ 减数。实际使用时，先比较大小决定符号和顺序：

```cpp
if (a < b) swap(a, b), sign = -1;  // 确保大减小
```

## 高精度乘法

```cpp
vector<int> mul(const vector<int>& a, const vector<int>& b) {
    vector<int> c(a.size() + b.size(), 0);  // 积最多 a.len + b.len 位
    for (int i = 0; i < a.size(); i++) {
        for (int j = 0; j < b.size(); j++) {
            c[i + j] += a[i] * b[j];         // 第 i 位 × 第 j 位 → 影响第 i+j 位
            if (c[i + j] >= 10) {            // 进位
                c[i + j + 1] += c[i + j] / 10;
                c[i + j] %= 10;
            }
        }
    }
    // 最后统一处理残留进位
    for (int i = 0; i + 1 < c.size(); i++) {
        if (c[i] >= 10) {
            c[i + 1] += c[i] / 10;
            c[i] %= 10;
        }
    }
    while (c.size() > 1 && c.back() == 0)    // 去除前导零
        c.pop_back();
    return c;
}
```

### 原理

手算竖式乘法的公式：a 的第 i 位（对应 $10^i$）乘 b 的第 j 位（对应 $10^j$），结果加到第 i+j 位（对应 $10^{i+j}$）。

```
      1  2  3
  ×      4  5
  -----------
      6  1  5    ← 123 × 5
    4  9  2      ← 123 × 40
  -----------
    5  5  3  5

关键: a[i] * b[j] → 贡献到 c[i+j]
```

```
i=0 (个位 3), j=1 (十位 4):
  3 × 4 = 12 → 加到 c[0+1] = c[1]
  对 10^0 × 10^1 = 10^1（十位）的贡献
  
i=2 (百位 1), j=0 (个位 5):
  1 × 5 = 5 → 加到 c[2+0] = c[2]
  对 10^2 × 10^0 = 10^2（百位）的贡献
```

### ⚠️ c 大小为 a.size() + b.size()

两个 n 位和 m 位的数相乘，结果最多 n+m 位。比如 99×99=9801（2 位×2 位=4 位）。

## 高精度除法（大数 ÷ int）

```cpp
vector<int> div(const vector<int>& a, int b, int& r) {
    r = 0;                                     // 余数
    vector<int> c(a.size(), 0);
    for (int i = a.size() - 1; i >= 0; i--) {  // ⚠️ 从高位往低位
        long long cur = r * 10LL + a[i];       // 当前被除数 = 上轮余数×10 + 当前位
        c[i] = cur / b;                        // 商
        r = cur % b;                           // 余数（传给下一轮）
    }
    while (c.size() > 1 && c.back() == 0)
        c.pop_back();
    return c;
}
```

### 原理

模拟手算长除法——从高位到低位逐位"拉下来"：

```
12345 ÷ 6：

        2 0 5 7  ← 商
     ┌─────────
   6 │ 1 2 3 4 5
       1 2       ← 1<6, 拉下2 → 12÷6=2 余 0
         0 3     ← 拉下3 → 3÷6=0 余3
           3 4   ← 拉下4 → 34÷6=5 余4
             4 5 ← 拉下5 → 45÷6=7 余3
               3 ← 最终余数
```

```
倒序数组 a = [5,4,3,2,1]

i=4: cur = 0×10 + 1 = 1  → c[4]=1/6=0,  r=1%6=1
i=3: cur = 1×10 + 2 = 12 → c[3]=12/6=2, r=12%6=0
i=2: cur = 0×10 + 3 = 3  → c[2]=3/6=0,  r=3%6=3
i=1: cur = 3×10 + 4 = 34 → c[1]=34/6=5, r=34%6=4
i=0: cur = 4×10 + 5 = 45 → c[0]=45/6=7, r=45%6=3

c = [7,5,0,2,0] → 去掉后导零 → 2057，余数 r=3
```

### 和加减乘的方向不同

- 加减乘：**从低位（i=0）往高位**循环
- 除法：**从高位（i=size-1）往低位**循环

因为除法手算时是从最高位开始逐位"拉下来"的。

## 辅助函数

```cpp
// 字符串 → 倒序 vector
vector<int> toBigInt(const string& s) {
    vector<int> a;
    for (int i = s.size() - 1; i >= 0; i--)
        a.push_back(s[i] - '0');
    return a;
}

// 倒序 vector → 字符串
string toString(const vector<int>& a) {
    string s;
    for (int i = a.size() - 1; i >= 0; i--)
        s += char(a[i] + '0');
    return s;
}
```

## 四种运算对比

| | 方向 | 核心操作 | 结果位数 |
|------|------|------|------|
| 加法 | 低→高 | `+ carry` | 最多 max(len)+1 |
| 减法 | 低→高 | `- borrow` | 最多 len（可能去前导零） |
| 乘法 | 低→高 | `c[i+j] += a[i]*b[j]` | 最多 lenA + lenB |
| 除法 | **高→低** | `cur = r*10 + a[i]` | 最多 len |

## ⚠️ 关键坑点

### 1. 存储方向：倒序

```
12345 → [5,4,3,2,1]
个位在 [0]，高位在末尾
```

输入反转一次，输出反转回来。方向搞反，加减乘除全错。

### 2. 去前导零

每次运算后都要 `while(size>1 && back()==0) pop_back()`，否则结果会多出无意义的零。

### 3. 除法的方向是反的

加减乘从 0 开始，除法从 size-1 开始。这是最常写错的地方。

### 4. 减法和除法需要额外处理

减法要保证大减小（否则要先交换，加负号）。除法只支持大数除以 int（大数除大数需要更复杂的算法）。

### 5. 乘法的进位处理

乘法进位有两种写法：边乘边进位（本文），或先全部加完再统一进位。统一进位更清晰但需要更大的中间数组。

## 总结

| 要点 | 说明 |
|------|------|
| 存储 | 倒序 vector，下标 0 是个位 |
| 加法 | 逐位加，满 10 进 1 |
| 减法 | 逐位减，不够向高位借 1 |
| 乘法 | `c[i+j] += a[i] × b[j]` |
| 除法 | 从高位往低位，`cur = r×10 + a[i]` |
| 去前导零 | 所有运算最后都要去 |
| 输入输出 | 存时反转，输出时反转回来 |
