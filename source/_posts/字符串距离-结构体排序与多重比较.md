## 题目描述

> 题目链接：[N诺/DreamJudge 1947](https://noobdream.com/DreamJudge/Issue/page/1947/)

字符串汉明距离：两个等长字符串不同字符位置的个数。给定 n 个字符串，输出前 6 对距离最小的 pair。排序优先级：距离小 → 较小字符串字典序小 → 较大字符串字典序小。

```
输入:
4
abcd
abcc
abdc
aaaa

输出:
abcd abcc 1
abcd abdc 2
abcc abdc 3
aaaa abcc 4
aaaa abdc 4
aaaa abcd 4
```

<!--more-->

这题算法本身不复杂，但写的过程中有几个之前一直不确定的知识点，值得专门记下来。

---

## 知识点一：结构体数组的定义和添加

### 定义结构体

```cpp
struct Node {
    string x;    // 字典序较小的字符串
    string y;    // 字典序较大的字符串
    int dis;     // 两者的汉明距离
};
```

### 放到 vector 里

```cpp
vector<Node> B;

// ✅ 最简洁（C++11 聚合初始化）
B.push_back({s1, s2, dis});

// 等价但啰嗦的写法
Node tmp;
tmp.x = s1; tmp.y = s2; tmp.dis = dis;
B.push_back(tmp);
```

竞赛中用 `push_back({s1, s2, dis})` 一行搞定，不用多写。

---

## 知识点二：多重条件的自定义排序

### 排序要求

```
1. dis 越小越靠前
2. dis 相同 → x（较小字符串）字典序越小越靠前
3. x 也相同 → y（较大字符串）字典序越小越靠前
```

### cmp 函数

```cpp
bool mycmp(const Node& a, const Node& b) {
    if (a.dis != b.dis) {
        return a.dis < b.dis;    // 第一优先级
    } else if (a.x != b.x) {
        return a.x < b.x;        // 第二优先级
    } else {
        return a.y < b.y;        // 第三优先级
    }
}

sort(B.begin(), B.end(), mycmp);
```

### 通用模板

```cpp
bool cmp(const T& a, const T& b) {
    if (a.f1 != b.f1) return a.f1 < b.f1;   // 优先级 1
    if (a.f2 != b.f2) return a.f2 < b.f2;   // 优先级 2
    if (a.f3 != b.f3) return a.f3 < b.f3;   // 优先级 3
    return false;  // 全相等，a 不排在 b 前
}
```

**核心**：每一级只在两个值**不等**时才返回比较结果，相等则交给下一级。`return true` 表示 a 排在 b 前面。

### ⚠️ 参数要用 `const Node&`

```cpp
// ✅ 传引用，不拷贝
bool mycmp(const Node& a, const Node& b) { ... }

// ❌ 传值，每次比较都拷贝 string
bool mycmp(Node a, Node b) { ... }
```

结构体里有 `string`，传值会拷贝整个字符串。`const &` 只传地址，快得多。

## 知识点三：swap 会破坏数组的原始顺序

### 问题的产生

遍历所有 pair 时，需要保证 `x` 是字典序较小的一方：

```cpp
for (int i = 0; i < n; i++) {
    for (int j = i + 1; j < n; j++) {
        if (A[i] > A[j]) {
            swap(A[i], A[j]);  // ❌ 改了原数组！
        }
        // A[i] 和 A[j] 的值变了，后面循环会乱
    }
}
```

`swap` 直接改了原数组。但外层循环依赖 `A[i]` 的值，内层循环依赖 `A[j]` 的遍历——原数组被改，后续 pair 组合可能漏掉或重复。

### 正确做法：在副本上 swap

```cpp
for (int i = 0; i < n; i++) {
    for (int j = i + 1; j < n; j++) {
        string s1 = A[i];  // 复制到局部变量
        string s2 = A[j];
        if (s1 > s2) {
            swap(s1, s2);  // ✅ 只交换副本
        }
        int dis = fun(s1, s2);
        B.push_back({s1, s2, dis});
    }
}
```

### 什么时候可以在原数组上 swap

只在**不需要保留原始顺序**的场景（比如只是输出结果、后续没有依赖原顺序的遍历）。只要后续有循环依赖原位置上的值，就不要动原数组。

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Node {
    string x;
    string y;
    int dis;
};

int fun(string x, string y) {
    int ans = 0;
    for (int i = 0; i < x.size(); i++) {
        if (x[i] != y[i]) ans++;
    }
    return ans;
}

bool mycmp(const Node& a, const Node& b) {
    if (a.dis != b.dis) {
        return a.dis < b.dis;
    } else if (a.x != b.x) {
        return a.x < b.x;
    } else {
        return a.y < b.y;
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    vector<string> A;
    for (int i = 0; i < n; i++) {
        string x;
        cin >> x;
        A.push_back(x);
    }

    vector<Node> B;
    for (int i = 0; i < n; i++) {
        for (int j = i + 1; j < n; j++) {
            string s1 = A[i], s2 = A[j];
            if (s1 > s2) swap(s1, s2);  // 在副本上交换
            int dis = fun(s1, s2);
            B.push_back({s1, s2, dis});
        }
    }

    sort(B.begin(), B.end(), mycmp);
    for (int i = 0; i < B.size() && i < 6; i++) {
        cout << B[i].x << " " << B[i].y << " " << B[i].dis << endl;
    }
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 结构体+vector | `vector<Node> B; B.push_back({x,y,dis});` |
| 多重排序 | `if (a.f1!=b.f1) return a.f1<b.f1;` 逐级比较 |
| cmp 参数 | 用 `const Node&` 传引用，避免拷贝 |
| swap 守则 | 只在局部副本上 swap，不动原数组 |
| 输出限制 | `i < B.size() && i < 6` |
