String 的构造方法

之前在c++里面 会让我产生一种String是和int double同样等位的类型 但是实际上在Java中 String是一种引用型数据类型 也就是说 String 是一种类 是可以创造对象 调用构造函数的

String 的创建对象通常是直接给一个字符串即可 非常方便 但是也有不同的构造函数

字符串常量池

字符串常量池中内的字符串一定是直接赋值的字符串 JDK7后的字符串常量池在堆内存中

当双引号直接赋值的时候 系统会检查字符串池中是否有“abc” 如果有直接吧地址给你 没有则新建
所以s1 和 s2 两个字符串其实指向的是同一个位置的abc

但是如果s1 和 s2不是由引号直接赋值 则不会出现这种情况 比如你分别new String(abc)

==号比较比的到底是什么？

  • 对于基本数据类型 比较的是数据值
  • 对于引用数据类型 比较的是地址值

对于只想比较字符串内容 我们是要用equals方法

所以 对于键盘输入的字符串 你肯定不能用==去比较 输入的字符串本质还是new 一个新的String

假想我们做用户登录见面 用户输入账户名 你要是用==来比较正确性 怎么都不可能true
