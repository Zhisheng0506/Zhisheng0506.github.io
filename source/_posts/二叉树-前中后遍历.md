## 题目描述

> 题目链接：[N诺/DreamJudge 1332](https://noobdream.com/DreamJudge/Issue/page/1332/)

输入 n 个整数，建立二叉排序树（BST），依次输出前序、中序、后序遍历结果。

```
输入:
5
3 1 4 2 5

输出:
3 1 2 4 5    ← 前序
1 2 3 4 5    ← 中序
2 1 5 4 3    ← 后序
```

<!--more-->

这题就是**模板题**——背熟模板、理解遍历顺序，直接默写。

---

## 树节点定义

```cpp
struct Node {
    int val;
    Node *l, *r;
    Node(int x) {
        val = x;
        l = r = nullptr;
    }
};
```

`l` 是左子树指针，`r` 是右子树指针。初始都为 `nullptr`。

---

## BST 插入

```cpp
void insert(Node* &root, int x) {
    if (root == nullptr) {
        root = new Node(x);    // 找到空位，插入
        return;
    }
    if (x < root->val) {
        insert(root->l, x);    // 小于 → 往左走
    } else if (x > root->val) {
        insert(root->r, x);    // 大于 → 往右走
    }
    // 相等：忽略（BST 不存重复值）
}
```

`Node* &root` 是**指针的引用**——加了 `&` 才能在递归中真正修改指针的值（把 `nullptr` 变成 `new Node(x)`）。不加 `&` 只改局部副本，树建不起来。

## 三种遍历

三种遍历的唯一区别是**访问根节点的时机**：

```
        3
       / \
      1   4
       \   \
        2   5
```

### 前序：根 → 左 → 右

```cpp
void preorder(Node* &root) {
    if (root == nullptr) return;
    cout << root->val << " ";   // 先根
    preorder(root->l);           // 再左
    preorder(root->r);           // 最后右
}
```

```
遍历: 3 → 1 → 2 → 4 → 5
输出: 3 1 2 4 5
```

### 中序：左 → 根 → 右

```cpp
void inorder(Node* &root) {
    if (root == nullptr) return;
    inorder(root->l);            // 先左
    cout << root->val << " ";   // 再根
    inorder(root->r);            // 最后右
}
```

```
遍历: 1 → 2 → 3 → 4 → 5
输出: 1 2 3 4 5    ← BST 中序 = 升序！
```

**BST 的中序遍历结果一定是从小到大有序的。** 这是机试中常用的验证技巧——如果中序不是升序，说明树建错了。

### 后序：左 → 右 → 根

```cpp
void postorder(Node* &root) {
    if (root == nullptr) return;
    postorder(root->l);          // 先左
    postorder(root->r);          // 再右
    cout << root->val << " ";   // 最后根
}
```

```
遍历: 2 → 1 → 5 → 4 → 3
输出: 2 1 5 4 3
```

## 三个模板对比

三份代码的差别只有 **`cout` 的位置**，其余完全相同：

```cpp
// 前序: cout 在最前
cout << val;  preorder(l);  preorder(r);

// 中序: cout 在中间
inorder(l);  cout << val;  inorder(r);

// 后序: cout 在最后
postorder(l);  postorder(r);  cout << val;
```

记忆口诀：
- 前序 = **根**左右
- 中序 = 左**根**右
- 后序 = 左右**根**

---

## ⚠️ 关键坑点

### 1. 指针引用 `Node* &root`

```cpp
void insert(Node* &root, int x)  // ✅ 有 &
void insert(Node* root, int x)   // ❌ 无 &，改了不生效
```

不加 `&`，`root = new Node(x)` 只改局部副本，调用者的指针还是 `nullptr`。树建完还是空的，遍历什么都不输出。

### 2. BST 相等值要忽略

```cpp
if (x < root->val) { ... }
else if (x > root->val) { ... }
// x == root->val: 跳过
```

漏了 `else if` 中的 `>` 判断，相等值可能死循环递归。

### 3. 遍历判空必须在第一行

```cpp
void preorder(Node* &root) {
    if (root == nullptr) return;  // ⚠️ 必须是第一行
    // ...
}
```

如果先访问 `root->val` 再判空，走到叶子的子节点时直接段错误。

### 4. 函数名和调用要一致

```cpp
preorder(root);    // 调用
preorder(root->l); // 递归
preorder(root->r);
```

三个位置要写**同一个函数名**。最常见的错误：写 `preorder` 递归内部又写了 `inorder`。

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Node {
    int val;
    Node *l, *r;
    Node(int x) { val = x; l = r = nullptr; }
};

void insert(Node* &root, int x) {
    if (root == nullptr) {
        root = new Node(x);
        return;
    }
    if (x < root->val) {
        insert(root->l, x);
    } else if (x > root->val) {
        insert(root->r, x);
    }
}

void preorder(Node* &root) {
    if (root == nullptr) return;
    cout << root->val << " ";
    preorder(root->l);
    preorder(root->r);
}

void inorder(Node* &root) {
    if (root == nullptr) return;
    inorder(root->l);
    cout << root->val << " ";
    inorder(root->r);
}

void postorder(Node* &root) {
    if (root == nullptr) return;
    postorder(root->l);
    postorder(root->r);
    cout << root->val << " ";
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    while (cin >> n) {
        Node* root = nullptr;
        for (int i = 0; i < n; i++) {
            int x; cin >> x;
            insert(root, x);
        }
        preorder(root);  cout << endl;
        inorder(root);   cout << endl;
        postorder(root); cout << endl;
    }
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 前序 | 根左右 — `cout → l → r` |
| 中序 | 左根右 — `l → cout → r`（BST 中序 = 有序） |
| 后序 | 左右根 — `l → r → cout` |
| 模板差异 | 只有 `cout` 位置不同 |
| 插入 | `Node* &root`（指针引用），小于往左大于往右 |
| 判空 | 遍历函数第一行必须 `if(root==nullptr) return` |
