## 一、闰年判断

### 规则

满足以下任一条件即为闰年：

1. 能被 400 整除
2. 能被 4 整除，但**不能**被 100 整除

```cpp
bool isLeap(int year) {
    return (year % 400 == 0) || (year % 4 == 0 && year % 100 != 0);
}
```

### 常见错误

- 把 `&&` 和 `||` 的优先级搞混，虽然这里 `&&` 优先于 `||` 刚好也是对的，但建议加括号，一眼不用想
- 只判断 `year % 4 == 0`，忘记整百年的例外（1900 年不是闰年）

### 闰年用于判断 2 月天数

用月份数组存每月天数，闰年改一下 2 月就行：

```cpp
int daysOfMonth[13] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};

if (isLeap(year)) {
    daysOfMonth[2] = 29;  // 闰年 2 月有 29 天
}
```

下标从 0 开始，0 号位置不用，月份直接当下标。

## 二、解析 YYYYMMDD 格式日期

机试中日期输入经常是 `20260525` 这种紧凑格式，用 `substr` 截取 + `stoi` 转数字来拆解。

### 基本用法

```cpp
string s;
cin >> s;  // s = "20260525"

int year  = stoi(s.substr(0, 4));   // "2026" → 2026
int month = stoi(s.substr(4, 2));   // "05"   → 5
int day   = stoi(s.substr(6, 2));   // "25"   → 25
```

### substr 详解

`substr(pos, len)` — 从位置 pos 开始，截取 len 个字符。

```cpp
string s = "20260525";

s.substr(0, 4);   // "2026"  从第 0 位开始截 4 个
s.substr(4, 2);   // "05"    从第 4 位开始截 2 个
s.substr(6, 2);   // "25"    从第 6 位开始截 2 个
s.substr(0);      // "20260525"  省略 len，截到尾
```

| 参数 | 含义 | 注意 |
|------|------|------|
| pos | 起始位置（从 0 开始） | 越界会抛 `std::out_of_range` |
| len | 截取长度（可省略） | 省略则截到末尾 |

### stoi 详解

`stoi(s)` — string to int，把字符串转成整数。

```cpp
int a = stoi("123");     // 123
int b = stoi("05");      // 5，前导零自动忽略
int c = stoi("-10");     // -10
int d = stoi("+42");     // 42
```

| 情况 | 行为 |
|------|------|
| 输入 `"12abc"` | 转换前缀 `12`，忽略后面 |
| 输入 `"abc"` | 抛 `std::invalid_argument` |
| 数字过大 | 抛 `std::out_of_range` |

### 相关函数一览

| 函数 | 方向 | 示例 |
|------|------|------|
| `stoi(s)` | string → int | `stoi("123")` = 123 |
| `stol(s)` | string → long | `stol("123")` = 123L |
| `stoll(s)` | string → long long | `stoll("123")` = 123LL |
| `stod(s)` | string → double | `stod("3.14")` = 3.14 |
| `to_string(n)` | 数字 → string | `to_string(123)` = "123" |

日期题最常用的是 `stoi` 和 `to_string` 这一对。

### ⚠️ 前导零的问题

`substr(4, 2)` 截出 `"05"`，`stoi("05")` 得 `5`。如果后续需要把月日拼回字符串做比较，前导零会丢失。需要保留时直接用 `substr` 结果，别来回转。

## 三、日期差值 — 从原点算起的"日差法"

> 题目链接：[N诺/DreamJudge 1096](https://noobdream.com/DreamJudge/Issue/page/1096/)

### 问题

给两个日期 YYYYMMDD，求它们之间相隔多少天（相邻两天算作相差 2 天）。

### 核心思想：统一原点

直接比较两个日期很麻烦——同年、跨年、跨世纪，情况太多。经典思路是：

> **分别算两个日期到"公元元年 1 月 1 日"的天数，然后相减。**

```text
date1 = 20110422
date2 = 20110412

daysFromOrigin(20110422) - daysFromOrigin(20110412)
= (从 1/1/1 到 date1 的天数) - (从 1/1/1 到 date2 的天数)
```

把"两个日期的差"转化为"两个原点距离的差"，问题简化成只写一个函数：**计算某个日期到原点 1/1/1 的总天数**。

### 为什么这样简化

求两个日期的差有三种思路：

| 方法 | 做法 | 评价 |
|------|------|------|
| 逐日累加 | 从 date1 一天天数到 date2 | 太慢，复杂度 O(天数) |
| 模拟进位 | 像手动算日期一样进退位 | 分支太多，容易写错 |
| **原点做差** | 分别算距 0/0/0 的天数再相减 | 代码最简洁，不需要讨论 |

### 代码实现

```cpp
int months[13] = {0, 31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};

bool isLeapYear(int year) {
    return (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);
}

int dateToDays(int year, int month, int day) {
    int total = 0;
    // 累加整年的天数
    for (int i = 1; i < year; i++) {
        if (isLeapYear(i)) total += 366;
        else total += 365;
    }
    // 累加当年整月的天数
    for (int m = 1; m < month; m++) {
        if (m == 2 && isLeapYear(year)) total += 29;
        else total += months[m];
    }
    total += day;
    return total;
}
```

### 主函数

```cpp
int main() {
    string beg, end;
    while (cin >> beg >> end) {
        int year1  = stoi(beg.substr(0, 4));
        int year2  = stoi(end.substr(0, 4));
        int month1 = stoi(beg.substr(4, 2));
        int month2 = stoi(end.substr(4, 2));
        int day1   = stoi(beg.substr(6, 2));
        int day2   = stoi(end.substr(6, 2));

        int sum1 = dateToDays(year1, month1, day1);
        int sum2 = dateToDays(year2, month2, day2);
        int ans  = abs(sum1 - sum2) + 1;  // ⚠️ 题目规定相邻差 2 天
        cout << ans << endl;
    }
}
```

### ⚠️ 注意事项

1. **闰年判断在循环里用** — 累加年份时每年都要判断，别漏了
2. **天数差 + 1** — 题目规定连续日期差值为 2 天（即 `dateToDays` 差值 + 1），不同题可能不同
3. **原点可以选 0/0/0 或 1/1/1** — 选哪个无所谓，做差时常数项会抵消

## 四、日期累加 — 进位模拟

> 题目链接：[N诺/DreamJudge 1421](https://noobdream.com/DreamJudge/Issue/page/1421/)

### 问题

给定日期和若干天，求加上天数后的新日期，输出格式 `yyyy-mm-dd`。

### 核心思想：逐层进位

和日期差值不同，这种题用原点法反而绕路。更直观的做法是**模拟进位**：

> 先把天数加上去，然后 while 超出当月就进月，月份超了就进年。

### 代码

```cpp
int daysofmonth(int year, int month) {
    int days[] = {31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31};
    if (month == 2 && isLeapYear(year)) return 29;
    return days[month - 1];
}

int main() {
    int m;
    cin >> m;
    while (m--) {
        int y, mth, d, add;
        cin >> y >> mth >> d >> add;
        d += add;                          // 1. 先把天数加上
        while (d > daysofmonth(y, mth)) {  // 2. 超出当月就进位
            d -= daysofmonth(y, mth);
            mth++;
            if (mth > 12) {                // 3. 月份超了进年
                mth = 1;
                y++;
            }
        }
        cout << y << "-"
             << setw(2) << setfill('0') << mth << "-"
             << setw(2) << setfill('0') << d << endl;
    }
}
```

### ⚠️ 输出格式的坑

要求 `yyyy-mm-dd`，月日不足两位要补 0：

```cpp
cout << setw(2) << setfill('0') << mth;   // "03" 而非 "3"
```

| 操作符 | 作用 | 粘性 |
| ------ | ------ | ------ |
| `setw(n)` | 字段宽度 n | **一次性**，只影响下一个输出 |
| `setfill(c)` | 用字符 c 填充 | 粘性，设置后一直有效 |

因为 `setw` 是一次性的，`mth` 和 `d` 前每次都要写。`setfill` 是粘性的，设一次就够。但如果后面不想补 0 了，记得 `setfill(' ')` 改回来。

### 两种日期题型对比

| 题型 | 方法 | 关键 |
|------|------|------|
| 日期差值 | 原点做差 | 算到 1/1/1 的天数再相减 |
| 日期累加 | 进位模拟 | 先加天数，while 逐月进位 |
