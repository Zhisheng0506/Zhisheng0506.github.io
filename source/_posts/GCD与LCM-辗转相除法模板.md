## 概述

最大公约数（GCD）和最小公倍数（LCM）是机试中的基础数学工具，本身不单独出题，但频繁作为**中间步骤**出现在约分、周期性判断、分数运算等场景。模板记熟就行。

<!--more-->

## GCD：辗转相除法

```cpp
int gcd(int a, int b) {
    while (b) {
        int t = a % b;
        a = b;
        b = t;
    }
    return a;
}
```

### 原理

```
gcd(a, b) = gcd(b, a % b)

重复直到 b = 0，此时 a 就是最大公约数。
```

### 手动演算

```
gcd(48, 18):
  48 % 18 = 12  → gcd(18, 12)
  18 % 12 = 6   → gcd(12, 6)
  12 % 6  = 0   → gcd(6, 0)
  返回 6
```

```
gcd(17, 5):
  17 % 5 = 2   → gcd(5, 2)
  5 % 2  = 1   → gcd(2, 1)
  2 % 1  = 0   → gcd(1, 0)
  返回 1（互质）
```

### 为什么叫"辗转相除"

反复用较小的数去除较大的数，余数替换原来的大数，**辗转**下去直到整除。

### C++17 的便捷写法

```cpp
int g = gcd(a, b);  // <numeric>，C++17
```

机试环境不稳定时自己写更安全，而且只有三行。

## LCM：先除后乘

```cpp
long long lcm(int a, int b) {
    return 1LL * a / gcd(a, b) * b;
}
```

### 公式

```
LCM(a, b) = a × b ÷ GCD(a, b)
```

### ⚠️ 为什么先除后乘

```cpp
// ❌ 危险写法：先乘后除，可能溢出
return a * b / gcd(a, b);
// a=50000, b=60000 → a*b=3×10^9 → int 溢出

// ✅ 安全写法：先除后乘
return 1LL * a / gcd(a, b) * b;
// 50000/2*60000 = 25000*60000 = 1.5×10^9 → long long 安全
```

**`a / gcd(a, b)` 一定是整数**（gcd 是 a 的约数），先除不会产生小数，又能把中间值压到最小。

### 为什么用 `1LL`

`1LL * a` 把运算提升到 `long long` 类型，后续所有乘法都在 64 位下进行，避免中间溢出。

```
a=100000, b=100000:
1LL*100000/100000*100000  →  1*100000  = 100000  (64位)
      int 运算则是 100000*100000=10^10 → 溢出
```

## 模板速查

```cpp
// 最大公约数
int gcd(int a, int b) {
    while (b) {
        int t = a % b;
        a = b;
        b = t;
    }
    return a;
}

// 最小公倍数
long long lcm(int a, int b) {
    return 1LL * a / gcd(a, b) * b;
}
```

## 常见使用场景

| 场景 | 用法 |
|------|------|
| 约分 | 分子分母同除 `gcd` |
| 分数通分 | 分母取 `lcm` |
| 周期性相遇 | 两个周期分别是 a 和 b，相遇周期 = `lcm(a, b)` |
| 判断互质 | `gcd(a, b) == 1` |
| 数组全体 gcd | 逐个 `g = gcd(g, A[i])` |

---

## ⚠️ 关键坑点

### 1. LCM 必须用 long long

```cpp
long long lcm(int a, int b) { ... }  // ✅ 返回值是 long long
int lcm(int a, int b) { ... }        // ❌ 即使先除后乘，结果也可能超 int
```

两个 ≤ 10^5 的 int，lcm 可能达到 10^10，`int` 装不下。

### 2. 先除后乘，不是先乘后除

即使返回值用 `long long`，如果先乘（`a * b`）在 int 下计算完才转，溢出已经发生了。

### 3. gcd 的 while 条件是 `b` 不是 `a % b`

```cpp
while (b) { ... }      // ✅ 当 b=0 时退出
while (a % b) { ... }  // ❌ 最后一轮 b 没赋值给 a
```

## 总结

| 要点 | 说明 |
|------|------|
| gcd | 辗转相除 `gcd(a,b) = gcd(b, a%b)`，while(b) 三行 |
| lcm | `a / gcd(a,b) * b`，先除后乘防溢出 |
| 1LL | 提升到 long long，避免中间溢出 |
| lcm 返回值 | 必须是 `long long` |
