## 什么是矩阵乘法

两个矩阵 A 和 B 相乘，要求 **A 的列数 = B 的行数**：

$$A_{a \times b} \times B_{b \times c} = C_{a \times c}$$

结果矩阵 C 的第 i 行第 j 列，等于 A 的第 i 行与 B 的第 j 列的**点积**：

$$C[i][j] = \sum_{k=0}^{b-1} A[i][k] \times B[k][j]$$

<!--more-->

## 核心代码

```cpp
for (int i = 0; i < a; i++)       // 遍历 C 的行
    for (int j = 0; j < c; j++)    // 遍历 C 的列
        for (int k = 0; k < b; k++) // A的列 = B的行 = 公共维度
            C[i][j] += A[i][k] * B[k][j];
```

这个循环顺序 **i-j-k** 对机试场景最友好：C 按行写入，`C[i][j]` 在内层循环中连续访问同一行的相邻列，缓存命中率高。

## ⚠️ 关键注意事项

### 1. C 必须初始化为 0

```cpp
vector<vector<int>> C(a, vector<int>(c, 0));  // ⚠️ 第三个参数 0 不能省
```

矩阵乘法是累加操作 `+=`，如果 C 不初始化，里面是随机值，结果全是错的。

### 2. 维度的"中间匹配"规则

```
A(a×b) × B(b×c) → C(a×c)
      ↑  b 必须相等
```

输入顺序是 `a b c`，B 的行数是 b，列数是 c。定义 B 时是 `B(b, vector<int>(c))`，别忘了 b 和 c 的位置。

### 3. k 循环的边界是 b，不是 a 或 c

```cpp
for (int k = 0; k < b; k++)   // 公共维度 = b = A的列数 = B的行数
```

一个常见的低级错误是把 k 的边界写成 a 或 c：方阵时三个值相等看不出来，矩形矩阵直接越界。

### 4. 循环顺序 i-k-j vs i-j-k

两种写法计算结果一样，但性能不同：

```cpp
// i-j-k：内层 k 循环累加到 C[i][j]，缓存友好
for (int i = 0; i < a; i++)
    for (int j = 0; j < c; j++)
        for (int k = 0; k < b; k++)
            C[i][j] += A[i][k] * B[k][j];

// i-k-j：内层 j 循环，A[i][k] 提出来，缓存更友好
for (int i = 0; i < a; i++)
    for (int k = 0; k < b; k++)
        for (int j = 0; j < c; j++)
            C[i][j] += A[i][k] * B[k][j];
```

机试题矩阵规模不大，i-j-k 就够。n≥500 时 i-k-j 快 2-3 倍。

### 5. 结果矩阵的大小是 a×c

输出时外层 `i < a`，内层 `j < c`，不是 b。

## 完整代码模板

```cpp
#include <iostream>
#include <vector>
using namespace std;

int main() {
    int a, b, c;
    cin >> a >> b >> c;

    vector<vector<int>> A(a, vector<int>(b));
    vector<vector<int>> B(b, vector<int>(c));
    vector<vector<int>> C(a, vector<int>(c, 0));  // ⚠️ 初始化为 0

    for (int i = 0; i < a; i++)
        for (int j = 0; j < b; j++)
            cin >> A[i][j];

    for (int i = 0; i < b; i++)
        for (int j = 0; j < c; j++)
            cin >> B[i][j];

    for (int i = 0; i < a; i++)
        for (int j = 0; j < c; j++)
            for (int k = 0; k < b; k++)
                C[i][j] += A[i][k] * B[k][j];

    for (int i = 0; i < a; i++) {
        for (int j = 0; j < c; j++)
            cout << C[i][j] << " ";
        cout << endl;
    }

    return 0;
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 维度规则 | A(a×b) × B(b×c) = C(a×c)，中间维度 b 必须相等 |
| C 初始化 | 必须全 0，否则 `+=` 结果全错 |
| k 的边界 | `k < b`，公共维度 |
| 循环写法 | 机试用 i-j-k，代码直观 |
| 输出大小 | a 行 c 列，不是 b |
