## 题目描述

> 题目链接：[N诺/DreamJudge 1396](https://noobdream.com/DreamJudge/Issue/page/1396/)

按顺序向初始为空的二叉排序树插入 N 个互不相同的关键字。每次插入成功后，输出**该节点的父节点关键字**。根节点输出 `-1`。

```
输入:
5
2 5 1 3 4

输出:
-1    ← 2 是根节点
2     ← 5 的父节点是 2
2     ← 1 的父节点是 2
5     ← 3 的父节点是 5
3     ← 4 的父节点是 3
```

<!--more-->

这是 [二叉树-前中后序遍历模板](/2026/06/04/二叉树-前中后遍历/) 的姊妹篇，侧重**插入时追踪父节点**这个技巧。

---

## 核心思路

在 BST 插入过程中，每到一个节点就把它的值作为"潜在父节点"，继续深入子树。最终插入位置的父亲，就是递归路径上最后一站的节点值。

```
树结构:
        2
       / \
      1   5
         /
        3
         \
          4
```

插入 4 的路径: 2→5→3→null，途径 2、5、3，最后一个非空节点是 3 → 父节点 = 3。

---

## 代码实现

```cpp
void insert(Node* &root, int x, int fa) {
    if (root == nullptr) {
        root = new Node(x);
        cout << fa << endl;   // ⚠️ 插入时输出父节点值
        return;
    }
    if (x < root->val) {
        insert(root->l, x, root->val);   // 当前节点成为下一层的"父亲"
    } else if (x > root->val) {
        insert(root->r, x, root->val);
    }
}
```

**`fa` 参数**：`insert(root->l, x, root->val)` 中，`root->val` 就是传给下一层的 `fa`——"你往下走，你就是你儿子的父亲"。

### 执行过程

```
输入: 2 5 1 3 4

根=2: root==null → new Node(2), cout << -1 （特判）

插入 5: insert(2, 5, fa=0)
  5>2 → insert(null, 5, fa=2)
  root==null → new Node(5), cout << 2 ✓

插入 1: insert(2, 1, fa=0)
  1<2 → insert(null, 1, fa=2)
  root==null → new Node(1), cout << 2 ✓

插入 3: insert(2, 3, fa=0)
  3>2 → insert(5, 3, fa=2)
  3<5 → insert(null, 3, fa=5)
  root==null → new Node(3), cout << 5 ✓

插入 4: insert(2, 4, fa=0)
  4>2 → insert(5, 4, fa=2)
  4<5 → insert(3, 4, fa=5)
  4>3 → insert(null, 4, fa=3)
  root==null → new Node(4), cout << 3 ✓
```

## 和上一篇 BST 的对比

| | 上一篇（遍历模板） | 本篇（父节点查找） |
|------|------|------|
| insert 参数 | `(root, x)` | `(root, x, fa)` |
| 插入时操作 | 只建节点 | 建节点 + 输出 fa |
| 根节点处理 | 直接插入 | 特判输出 -1 |
| 用途 | 建树 + 遍历 | 建树 + 追踪父亲 |

核心差异只多了一个 `fa` 参数，模板几乎一样。

### 为什么根节点要特判

```cpp
if (root == nullptr) {
    root = new Node(x);
    cout << "-1" << endl;  // 根节点单独处理
} else {
    insert(root, x, 0);    // fa=0 只是占位，不会被用到
}
```

根的父亲是 `-1`，必须单独判断。`insert` 的 `fa` 初始值随便填，因为第一次走到 `nullptr` 之前一定会经过至少一层递归，`fa` 会被覆盖为真正的节点值。

---

## ⚠️ 关键坑点

### 1. 根节点必须单独处理

```cpp
// ✅
if (root == nullptr) {
    root = new Node(x);
    cout << "-1" << endl;
} else {
    insert(root, x, 0);
}

// ❌ 所有节点统一走 insert
insert(root, x, -1);
// fa=-1 传给根，但根的父节点不是 -1 而是"不存在"
// 而且 insert 内部没法区分"这是根"还是"这是真的父亲"
```

### 2. fa 的传递时机

```cpp
insert(root->l, x, root->val);
//                 ^^^^^^^^^
//                 自己成为下一层的父亲
```

`fa` 就是当前递归层的节点值——往下走之前，把自己传给下一层。

### 3. `Node* &root` 必须加 `&`

和上一篇一样，不加 `&` 树建不起来，后续所有插入的父节点都找错。

## 完整代码模板

```cpp
#include <bits/stdc++.h>
using namespace std;

struct Node {
    int val;
    Node *l, *r;
    Node(int x) {
        val = x;
        l = r = nullptr;
    }
};

void insert(Node* &root, int x, int fa) {
    if (root == nullptr) {
        root = new Node(x);
        cout << fa << endl;
        return;
    }
    if (x < root->val) {
        insert(root->l, x, root->val);
    } else if (x > root->val) {
        insert(root->r, x, root->val);
    }
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);
    int n;
    while (cin >> n) {
        Node* root = nullptr;
        for (int i = 0; i < n; i++) {
            int x; cin >> x;
            if (root == nullptr) {
                root = new Node(x);
                cout << "-1" << endl;
            } else {
                insert(root, x, 0);
            }
        }
    }
}
```

## 总结

| 要点 | 说明 |
|------|------|
| fa 参数 | 当前节点的值，传给下一层递归作为"父亲" |
| fa 传递 | `insert(root->l, x, root->val)` |
| 根节点 | 特判输出 `-1`，不走 `insert` |
| 和上一篇差异 | 只多了一个 `fa` 参数 |
| 其余相同 | `Node* &root`、小于往左大于往右 |
